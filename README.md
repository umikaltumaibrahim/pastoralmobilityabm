# pastoralmobilityabm

This repository contains the model code for the framework that integrates Bayesian optimization, Monte Carlo robustness analysis, and Approximate Bayesian Computation (ABC) for calibration, validation and uncertainty analysis supporting the manuscript: "A Bayesian Optimization and Approximate Bayesian Computation Framework for Calibrating Agent-Based Models of Pastoralist Mobility in Dryland Systems"

## Contents

- Repast Simphony model
- Calibration, Validation and Uncertainty Analysis scripts (Bayesian Optimization, Monte Carlo, ABC)

## Data Sources

See manuscript for full citations.

## Software Requirements

- Repast Simphony 2.11.0
- Java 17
- Python 3.1.3
